# 🎨 Website Visual Preview

**Bold, energetic, playful** - Designed to make near-expiry snacks feel like exclusive treasure hunting.

🎨 **Supporting:**
- Cream (#FFF5E1) - Text on dark

- **Bebas Neue** - Headers (tall, condensed, impactful)

## Interactive Elements

### 🖱️ Hover Effects:
- **CTA Buttons**: Lift effect (translate shadow)
- **Feature Cards**: Translate up + gradient border glow
- **Pricing Cards**: Extra lift + border color change
- **Plan Buttons**: Background fills with cyan

### 🎭 Animations:
- **Page Load**: Staggered text reveals (logo → title → subtitle → CTA)
- **Floating Items**: Gentle continuous float and rotate
- **Background**: Subtle pulsing gradient
- **Scroll**: Sections fade in as you scroll

### 📱 Responsive Behavior:
- **Desktop** (1200px+): All sections full width, 3-column grids
- **Tablet** (768px-1199px): 2-column grids, slightly smaller text
- **Mobile** (<768px): Single column, stacked layout, adjusted typography

---

## Design Philosophy

**What Makes It Distinctive:**
1. **Bold Typography**: Huge, in-your-face text that demands attention
2. **High Contrast**: Dark background with vibrant accents
3. **Playful Energy**: Floating emojis, unexpected color combos
4. **Premium Budget Feel**: Makes "near-expiry" feel exclusive
5. **Geometric Clarity**: Sharp angles, no rounded corners (except where necessary)
6. **Shadow Play**: Heavy, offset shadows for depth

**Avoids:**
- ❌ Generic gradients (purple on white)
- ❌ Overused fonts (Inter, Roboto)
- ❌ Cookie-cutter layouts
- ❌ Timid color choices
- ❌ Boring, safe design

**Embraces:**
- ✅ Confident, bold aesthetics
- ✅ Unexpected font pairings
- ✅ High-energy color combinations
- ✅ Purposeful chaos
- ✅ Memorable visual identity

## Technical Details

**CSS Approach:**
- Scoped styles with `<style jsx>`
- CSS custom properties (variables) for colors
- CSS animations (no JavaScript required for most effects)
- Mobile-first responsive design
- Flexbox and Grid for layouts

**Performance:**
- Google Fonts loaded async
- CSS animations use transform/opacity (GPU accelerated)
- No heavy images (emoji-based icons)
- Minimal JavaScript (only for form handling and GA)

**Accessibility:**
- High contrast ratios
- Focus states on interactive elements
- Semantic HTML structure
- Responsive font sizing (clamp)

## User Journey

1. **Land on Hero** → Immediately grabbed by bold text and playful vibe
2. **Scroll to Features** → Understand value props quickly (icons + short text)
3. **Browse Pricing** → See options, "Most Popular" draws eye to middle tier
4. **Select Plan** → Interactive selection gives satisfaction
5. **Sign Up** → Simple, clear form at bottom
6. **Done** → Memorable experience, likely to share
